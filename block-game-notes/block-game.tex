\section*{Representing the world}
\paragraph*{how to: represent as meshes?}
represent the world as meshes that we can update per frame

\paragraph*{chunk updates}
the solution is to separate the meshes into chunks. when a chunk is
updated, we update the corresponding mesh, as opposed to updating the
whole world as one mesh

\paragraph*{chunk size?}
maybe 16 voxels wide as tribute to minecraft. we could probably optimize this

\paragraph*{how to: represent individual voxels?}
